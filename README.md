# BrightLane Affiliate Content Site

BrightLane is an informational affiliate content website focused on financial education topics and loan-related search intent.

## 🌐 Live Site
https://brightlane.github.io/BrightLane/

---

## 📌 Project Purpose

This site is designed to:
- Provide informational content about personal finance topics
- Target high-intent search queries (SEO traffic)
- Route users through informational pages to a single conversion endpoint

---
